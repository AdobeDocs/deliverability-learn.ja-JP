# インターネットサービスプロバイダーの詳細

ISPはそれぞれ異なります。 利用者の受信トレイに送る電子メールを決定する際や、迷惑メールフォルダーにフィルタリングされる電子メールを決定する際、または電子メールを全く通さない電子メールを決定する際には、さまざまな点に重点を置きます。

ここからは、ISPの一部の重要な違いについて簡単に説明します。 ただし、すべてのISPを網羅したり、考えられるすべての違いを網羅したりすることを意図していません。
