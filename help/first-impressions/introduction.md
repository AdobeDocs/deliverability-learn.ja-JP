# ファーストインプレッション - リスト収集とウェルカムメール

新しい関係を築くためには、優れた第一印象を与えることが重要です。 それがなければ、あなたが言ったり行ったりすることは何もチャンスを与えられないかもしれません。 企業と、その企業とインタラクションすることを選択した企業との関係も同じです。 すべてはどこかから始まります。 メールプログラムの実行は、メールアドレスを収集し、登録者を歓迎することから始まります。 メールプログラムで優れた第一印象を与えて、成功を収めるための準備を整える方法を見てみましょう。

手順1: [ アドレス収集とリストの増加](/help/first-impressions/address-collection-and-list-growth.md):
*新しい電子メールアドレスに最適なソース、高いデータ品質を確保する方法、法的ガイドラインとの整合性について説明します。*

ステップ 2: [ ウェルカムメール ](/help/first-impressions/welcome-emails.md):
*ウェルカムメールを作成する際に考慮すべき重要な要素について理解します。*
