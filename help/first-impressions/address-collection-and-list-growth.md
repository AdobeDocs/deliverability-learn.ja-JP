# アドレス収集とリスト拡大

新しいメールアドレスの最適なソースは、web サイトや実店舗でのサインアップなどの直接ソースです。 このような状況では、エクスペリエンスを制御して、肯定的であり、購読者がブランドからのメールの受信に興味があることを確認できます。

これらのサインアップ方法に関する注意事項：

**物理的な保存** リストの収集では、口頭または書面によるアドレス入力が原因で、アドレスのスペルミスが発生し、課題が生じる場合があります。 店舗でのサインアップ後、できるだけ早く確認メールを送信することをお勧めします。

最も一般的な形式の **web サイトへのサインアップ** は、「シングルオプトイン」です。 これは、メールアドレスを取得するために使用する必要がある絶対的な最小標準です。 単一オプトインとは、特定のメールアドレスの所有者が送信者にマーケティングメールを送信する権限を付与する場合のことで、通常は web フォームや店舗へのサインアップを使用してアドレスを送信します。 この方法を使用してメールキャンペーンを正常に実行することは可能ですが、問題の原因になる場合があります。

* 未確認のメールアドレスには、入力ミスがある、形式が正しくない、正しくない、悪意をもって使用される、などの可能性があります。 入力ミスや不正なアドレスは高いバウンス率の原因となり、ISP が発行したブロックを誘発したり、IP のレピュテーションが失われたりする可能性があります。

* 既知のスパムトラップを悪意を持って送信すると（「リスト中毒」とも呼ばれます）、そのトラップの所有者がアクションを実行した場合、配信および評判に関して大きな問題が発生する可能性があります。 受信者が、確認を行わずにマーケティングリストに本当に追加したいかどうかを把握することは不可能です。 これにより、受信者の期待値を設定できなくなるので、スパムの苦情が増える可能性があります。また、収集されたメールがスパムトラップであるかどうかをブロックリストに加えるする場合もあります。

実店舗とシングルオプトインの両方で発生する問題を最小限に抑える方法のガイダンスについては、このガイドの [ データ品質とハイジーン ](#data-quality-and-hygiene) の節を参照し、ダブルオプトインの詳細とメリットを確認してください。

>[!NOTE]
>
>購読者は、web サイトから目的の情報を取得するだけでなく、マーケティングリストに追加されないようにするために、不要なアドレス、期限切れのアドレス、または自分とは異なるアドレスを使用することがよくあります。 これが発生すると、マーケターのリストに含まれるハードバウンスの数が多く、スパムの苦情率が高く、クリックしたり、メールを開いたり、積極的にメールに関与したりしない購読者が増えます。 これは、メールボックスプロバイダーや ISP にとっては危険信号と見なすことができます。

## サインアップフォーム

データのフィールドを追加する以外に、新しい購読者に関する収集が必要な情報として、Web サイト上には、サインアップフォームに関して行うべきことがいくつかあります。

* メールの受信に同意していること、何を受信するか、どのくらいの頻度で受信するかを購読者に明確に伝えます。
* サブスクライバーが受信する通信の頻度またはタイプを選択できるようにするオプションを追加します。 これらのオプションを使用すると、購読者の好みを最初から把握できるので、新規顧客に最適なエクスペリエンスを提供できます。
* 登録プロセス中に購読者の興味を失うリスクのバランスを取るには、できるだけ多くの情報を求めます。 誕生日、場所、興味などをカスタマイズして、よりカスタマイズされたコンテンツを送信できます。 すべてのブランドの購読者には異なる期待と許容値のしきい値があるので、テストは、状況に合った適切なバランスを見つけるための鍵です。

>[!NOTE]
>
> サインアッププロセス中は、事前にチェックボックスを使用しないでください。 法的にトラブルに巻き込まれることもありますが、これも否定的な顧客体験です。

## データ品質とハイジーン

データの収集は、課題の一部にすぎません。 また、データが正確で使用可能であることも確認する必要があります。 基本的な形式フィルターを配置する必要があります。 「@」や「。」が含まれていないメールアドレスは無効です。 例： 一般的なエイリアスアドレスは許可しないでください。これらは役割アカウントとも呼ばれます（「情報」、「管理者」、「営業」、「サポート」など）。 役割アカウントは、その性質上、受信者に 1 人の購読者ではなくグループの人物が含まれているため、リスクが生じる可能性があります。 グループ内で期待値と許容値が異なり、それにより苦情、様々な契約、購読解除、一般的な混乱が生じるリスクが高まる可能性があります。

メールアドレスのデータで発生する可能性のある一般的な問題に対する解決策を次に示します。

**[!DNL Double opt-in (DOI)]**
[!DNL Double opt-in (DOI)] は、ほとんどのメールの専門家にとって、最高の配信品質プラクティスと見なされています。 DOI は、お客様のウェルカムメールでスパムトラップや苦情に問題がある場合、メールを受信する購読者が実際にメールプログラムに登録し、メールの受信を希望していることを確認する優れた方法です。

DOI は、メールプログラムにサインアップした購読者のメールアドレスに確認メールを送信します。このアドレスには、同意を確認するためにクリックする必要があるリンクが含まれています。 この獲得方法では、購読者が確認しない場合、送信者はより多くのメールを送信しません。 新しい購読者に web サイト上でこれを実行していることを知らせ、続行する前にサインアップを完了するように促します。 この方法では、サインアップ数は減少しますが、サインアップを行う人々は非常に関与が高く、長期的に滞在する傾向があります。 通常、送信者の ROI がはるかに高くなります。

**非表示のフィールド**
サインアップフォームに非表示のフィールドを適用することは、自動ボットのサインアップを実際の購読者と区別する優れた方法です。 データフィールドはHTMLコードでは表示されず、非表示になっているので、ボットは人間が入力できないデータを入力します。このメソッドを使用すると、非表示のフィールドに入力されたデータを含むサインアップを抑制するルールを作成できます。

**[!DNL re-CAPTCHA] は、購読者が実際の人物ではなくボットである可能性を減らすために使用できる検証方法です。 様々なバージョンがあり、一部にはキーワード識別または画像が含まれています。 一部のバージョンは他のバージョンよりも効果的で、セキュリティと配信品質の問題の防止で得られるメリットは、コンバージョンに対する悪影響よりもはるかに大きくなります。

## 法律ガイドライン

弁護士に相談して、電子メールに関する地域および国の法律を解釈してもらいます。 メールの法律は、国内の国や地域によって大きく異なることに注意が必要です。

* 購読者の国の法律に準拠するように、購読者の位置情報を必ず収集してください。 その詳細がなければ、加入者に市場を出す方法に制限することができます。
* 関連する法律は、送信者ではなく、受信者の場所によって決定されます。 だから、あなたはあなたが加入者を持つことができるすべての国のために法律を知って従わなければなりません。
* 加入者の居住国を完全に確実に知ることは困難です。 顧客から提供されるデータは古くなっている可能性があり、Gmail や Yahoo のように VPN や画像ウェアハウジングが原因で、ピクセルの場所データが不正確になる場合があります。 確信がない場合は、可能な限り厳格な法律およびガイドラインを適用するのが最も安全です。

## その他の推奨されないリスト収集方法

アドレスを収集する方法は他にも多数あり、それぞれに独自の機会、課題、欠点があります。 多くの場合、使用はプロバイダーの許容可能な使用ポリシーによって制限されるので、Adobeは一般的にこれらを推奨しません。 リスクを制限または回避する際に役立つ危険性を学ぶために、いくつかの一般的な例を見ていきます。

**リストの購入またはレンタル**
メールアドレスには様々な種類があります。 例えば、プライマリ用メール、仕事用メール、学校用メール、セカンダリメール、非アクティブなメールなどです。 購入リストやレンタルリストを通じて収集され、共有されるアドレスの種類は、プライマリのメールアカウントはほとんどありません。このようなアカウントでは、ほぼすべてのエンゲージメントや購入アクティビティが行われます。

運が良ければ、セカンダリーアカウントを手に入れます。セカンダリーアカウントでは、人々が何かを購入する準備ができたときに、お得な情報やオファーを探します。 これにより、通常、エンゲージメントレベルが低下します（存在する場合）。 運が悪ければ、リストは非アクティブなメールでいっぱいになり、スパムトラップになる可能性があります。 多くの場合、セカンダリメールと非アクティブなメールの両方が混在します。 一般に、これらのタイプのリストの品質は、電子メールプログラムにとって害よりも害を及ぼします。 これは、[Adobe Campaignの利用規定 ](https://www.adobe.com/jp/legal/terms/aup.html) により禁止されています。

**リストを追加**
これらは、ブランドとのエンゲージメントを選択した顧客です。これは素晴らしいことです。 しかし、彼らはメール以外の方法（店舗、ソーシャルメディアなど）を通じてエンゲージメントを行うことを選択しました。 お客様からリクエストのないメールを受け取ることに対して反応がなかっただけでなく、メールアドレスを提供しなかったので、お客様のメールアドレスの取得方法にも懸念を示している可能性があります。 この方法では、ブランドに関与した顧客または潜在的な顧客を、ブランドを信頼しなくなり、代わりに競争に行くデトラクターに変えるリスクがあります。 これは、[Adobe Campaignの利用規定 ](https://www.adobe.com/jp/legal/terms/aup.html) により禁止されています。

**展示会その他のイベントのコレクション**
ブースまたは別の公式を通じてアドレスを収集すると、明確にブランド化された方法が役立ちます。 このような多くのイベントでは、すべてのアドレスが収集され、イベントのプロモーターまたはホストを通じて配布されるリスクがあります。 つまり、これらのメールアドレスの所有者は、ブランドからのメールの受信をリクエストしたことはありません。 購読者はメールに苦情を言ったり、スパムとマークしたりする傾向があり、正確な連絡先情報を提供しなかった可能性があります。

**感嘆符**

広告キャンペーンは、大量のメールアドレスを素早く提供します。 しかし、これらの購読者は、あなたのメールではなく、賞金を望んでいます。 彼らは彼らに手を差し伸べる名前に注意を払っていなかったかもしれません。 彼らは苦情を言い、メールをスパムとマークする可能性が高く、関与したり購入したりする可能性はほとんどありません。
