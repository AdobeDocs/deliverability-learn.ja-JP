# 登録解除と登録解除

登録解除は、メールライフサイクルの自然な流れです。 しかし、マーケターが別れを告げるのは難しいことです。 最も魅力的なメールを作成し、自社が提供する製品やサービスに関する最高のお得な情報を提供することに尽力しています。 しかし、それは完全に正常であり、一部の加入者が進むことを決定することを期待すべきです。 この原因はいくつか考えられますが、重要なのは、顧客が企業からのメール配信に興味を示さなくなったことです。

メールリストの購読解除を管理する際に避けるべきことと避けるべきことのリストを作成しました。詳しくは、この記事をご覧ください。

## dos

**DO:**&#x200B;登録解除リンクが目立つように配置されています。 これは、ユーザーが「これはスパムです」という苦情（フィードバックループとも呼ばれます）をクリックするのを防ぐのに役立ちます。

+ 登録解除リンクを際立たせる方法に関するアイデアをいくつか紹介します。中央に配置し、下線を引き、太字にします。 電子メールの上部または下部に配置します。 別のフォントカラーにします。 ユーザーの言語設定で「登録解除」という単語を使用します。 過剰な表現や過剰な表現は避けましょう。

**DO:**&#x200B;購読解除リンク ドライブを、ユーザーがリストから削除することを確認できるランディングページに移動します。

ランディングページの言葉遣いには、「購読解除を確認してください」というクリックするためのリンクと、「オプトダウンして、あなたから受け取るメールの頻度を下げましょう」という別のリンクを含める必要があります。 これは、ワンクリックで登録解除できる方法ではありませんが、ボットクリックで誤った登録解除が発生することを防ぐのに役立ちます。

+ 登録解除によってオーディエンスがランディングページに移動する場合は、オーディエンスがランディングページにアクセスしたら、ワンクリックで登録解除できる機能を備えているかどうかを確認しましょう。

**DO:** プロセスを簡単かつ透明にします。 例：

+ 「これをクリックすると、すべてのマーケティングメールの購読を解除できます」。 これは、ユーザーに与えられる最初のオプションです。

+ 「これをクリックして環境設定を変更します」。

**DO:**&#x200B;登録解除ランディングページにメールアドレスを含めます。

+ 重要：ユーザーの場所/管轄区域によっては、完全なメールアドレスを含めることに関するプライバシー上の懸念がある場合があります。メールアドレスのハッシュ化されたバージョンを事前入力する必要がある場合があります。 これは、メールが最初に購読したアドレスから転送された可能性があるためです。 特定の要件について、法務グループと相談しながら把握しましょう。

**DO:**&#x200B;購読解除が正常に送信されたことを確認し、メールの受信を停止できるタイミングを確認します。

+ すでに購読解除しているユーザーが再試行する場合は、すでに購読解除されている日時を知らせます。

**DO:**&#x200B;登録解除を尊重し、アドレスをタイムリーに削除します。

+ 適時性は地域の法律と法域によって異なるため、ビジネスに適用される内容について弁護士に相談してください。

**DO:**&#x200B;購読者が関係を指示できるようにします。

+ 加入者にオプションを提供することも可能です。これは、プリファレンスセンター/オプトダウンの取り組みを通じて管理できます。

**DO:**&#x200B;電子メールテンプレートでリスト登録解除ヘッダーを使用すると、電子メールの上部に登録解除リンクを表示する電子メールクライアント（例：gmail.com）が、ユーザーが登録解除を探すのに慣れている場所で簡単に見つかります。


## Do nots


**禁止：**&#x200B;購読解除リンクをコンテンツに埋め込み、購読者が不満を感じて「諦める」ことを期待します。

+ その場合、エクスペリエンスが不十分になり、苦情が増加し、配信品質や評判に悪影響を及ぼす可能性があります。

**登録解除しない：**&#x200B;お客様がフォームにアクセスしたら、登録解除を難しくします。 詐欺は迷惑であり、場合によっては違法です。

+ エンドユーザーに誤解を与えないように直接的に対応する。

**登録解除フォームに電子メールアドレスを手動で追加しないでください：**。

+ 購読したアドレスは、ユーザーがメール用にログインしたアドレス（転送済みなど）とは異なる場合があります。 最初のサブスクリプションに使用されたメールアドレスを知らない可能性があります。

**購読を解除するには、アカウントにログインしてください。**

+ 間違ったアドレスにメールを送信している可能性があり、そのアドレス所有者はアカウントにアクセスできない可能性があります。 メールを受信している場合は、メール内のリンクから購読を解除することを許可する必要があります。他のアクションや情報は必要ありません。
+ 顧客が取るステップは少なければ少ないほど好都合です。

**登録解除を確認する電子メールを送信しないでください：**。

+ もしそうなら、あなたはすでに彼らにメールを送らないという約束を破っています。

**次の操作を行わない：**&#x200B;言葉遣いに気を配る。 「お得な情報は好きではありません。登録を解除してください！」

+ これは失礼なことに見えてくる可能性がある。 ネガティブなトーンを避ける：

**実行しない：** リクエストを処理する前に、購読解除の理由を強制的に提供します。

+ この情報を追加する場所を顧客に許可することは、将来のインサイトに関しては役立ちますが、「配信停止」ランディングページのオプションフィールドにすることができます。

+ より優れたアプローチは、登録解除に成功したことを伝え、その理由をオプションで提供するスペースを提供することです。

登録解除は、苦情よりも優れています。 メールプログラムに積極的にエンゲージしない人々は、配信品質のレピュテーションだけでなく指標にも悪影響を与える可能性があります。 今こそ、クリックして購入するまで待てない、魅力的で関連性の高いメッセージを配信することで、現在のオーディエンスとより深い関係を築くことに集中しましょう。
